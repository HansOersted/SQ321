# SQ321 - Singapore Airlines Flight Incident

This repository contains data related to the [SQ321 accident](https://www.flightradar24.com/blog/flight-tracking-news/major-incident/1-dead-dozens-injured-in-sq321-turbulence/) involving Singapore Airlines. The incident occurred during a flight from London Heathrow to Singapore Changi, where the aircraft encountered extreme turbulence, leading to serious injuries and one fatality.

## Incident Overview
- **Flight**: SQ321, Singapore Airlines
- **Date**: May 21, 2024
- **Aircraft**: Boeing 777-300ER (9V-SWM)
- **Location**: Over Myanmar, at an altitude of approximately 37,000 feet
- **Casualties**: 1 fatality, dozens injured

### Key Data
- [Link to the flight tracking page on Flightradar24](https://www.flightradar24.com/blog/flight-tracking-news/major-incident/1-dead-dozens-injured-in-sq321-turbulence/)
- **Type of Turbulence**: Extreme, clear-air turbulence
- **Vertical Drop**: Approximately 178 feet in 4.6 seconds
- **Investigation**: Preliminary reports from the Singapore Transport Safety Investigation Bureau (TSIB)

This repository includes flight and turbulence data, which can be used for further analysis, such as flight stability, turbulence impact, and control system robustness.
